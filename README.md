# `$ ship-release`

Publish new versions on GitHub and npm with ease.




## Installation

```sh
$ npm i ship-release
```



### CLI Usage
You can install the package globally and use it as command line tool:

```sh
$ npm i -g ship-release
```

Then, run `ship-release --help` and see what the CLI tool can do.
